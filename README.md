## 🎓 THE 13 INTELLIGENCE MODULES

### What Each Module Does

**Module 1: Improved Summary**
- Analyzes current professional summary
- Generates ATS-optimized version
- Extracts top keywords
- Scores compliance

**Module 2: Enhanced Experience**
- Detects weak action verbs
- Extracts metrics (%, $, numbers)
- Rewrites bullets with power verbs
- Calculates impact score

**Module 3: Skill Optimization**
- Detects current skills
- Maps to industry standards
- Identifies skill gaps
- Suggests trending skills to add

**Module 4: Multi-Dimensional Scores**
- Calculates 6 key scores:
  - ATS Score (0-100)
  - Recruiter Score (0-100)
  - Impact Score (0-100)
  - Skill Depth Score (0-100)
  - Career Consistency (0-100)
  - Overall Score (0-100)

**Module 5: Recruiter Simulation**
- Simulates recruiter's 6-second scan
- Provides first impression text
- Calculates shortlist probability %
- Lists key strengths & concerns

**Module 6: Resume Breakdown**
- Analyzes each bullet point
- Identifies generic language
- Shows improvement suggestions
- Calculates genericity percentage

**Module 7: Skill Intelligence**
- Categorizes skills:
  - Core skills (most relevant)
  - Weak skills (less emphasized)
  - Missing skills (gaps)
  - Future skills (trending)
- Calculates skill gap percentage

**Module 8: Career Insights**
- Calculates role alignment %
- Suggests 3-5 career paths
- Provides growth recommendations
- Estimates career timeline

**Module 9: Risk Detection**
- Identifies red flags
- Detects employment gaps
- Finds skill mismatches
- Sets severity level (Low/Med/High)

**Module 10: Interview Questions**
- Generates 5 technical questions
- Generates 5 HR questions
- Generates 3 situational questions
- Tailors to resume content

**Module 11: Personal Branding**
- Creates LinkedIn headline
- Writes professional tagline
- Composes short bio
- Crafts 30-second elevator pitch

**Module 12: Portfolio Content**
- Generates hero section text
- Writes about section
- Creates skills introduction
- Lists portfolio projects
- Writes call-to-action

**Module 13: Optimized Resume**
- Fully rewrites entire resume
- Applies ATS best practices
- Tailors to job description (if provided)
- Calculates job match percentage

## 🚀 QUICK START (3 Steps)

### Step 1: Install
```bash
# Backend dependencies
pip install fastapi pydantic openai python-multipart

# Frontend dependencies
cd frontend && npm install
```

### Step 2: Configure
```bash
export OPENAI_API_KEY=sk-your-key
export JWT_SECRET=your-secret
```

### Step 3: Run
```bash
# Terminal 1: Backend
python -m uvicorn backend.app.main:app --reload

# Terminal 2: Frontend
cd frontend && npm run dev

# Visit: http://localhost:3000/resume
```
